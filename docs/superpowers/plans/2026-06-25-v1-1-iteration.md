# DiffGuard V1.1 实施计划

> **给 agentic workers：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 按任务逐步执行本计划。步骤使用 checkbox（`- [ ]`）语法跟踪。

**目标：** 通过更丰富的 findings、provider presets、连接测试和结果面板操作，提升首个版本的可用性。

**架构：** 保持当前 Review pipeline 不变。为 findings 添加可选元数据，保持 JSON 解析向后兼容，引入小型 settings/provider helper，并用过滤、复制、重新运行回调扩展现有 Swing 结果面板。

**技术栈：** Kotlin、IntelliJ Platform Swing UI DSL、kotlinx.serialization、OkHttp、JUnit 5。
